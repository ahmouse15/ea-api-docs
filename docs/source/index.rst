Documentation for EA APIs!
===================================

This project aims to document EA APIs used for login and management/downloading of games.

Check out the :doc:`api` section for further information, including
how to :ref:`usage` the project.

.. note::

   These are unofficial docs. EA may change their API at any time without notice, potentially leaving these docs out of date.
